Adds a moderator to the broadcaster’s moderation room. add-channel-moderator

Accessing the endpoint

Request: AddChannelModeratorRequest

To use this endpoint, construct a AddChannelModeratorRequest with the AddChannelModeratorRequest::builder() method.

use twitch_api::helix::moderation::add_channel_moderator;
let request = add_channel_moderator::AddChannelModeratorRequest::builder()
    .broadcaster_id("1234")
    .moderator_id("5678")
    .build();

Response: AddChannelModeratorResponse

Send the request to receive the response with HelixClient::req_post().

use twitch_api::helix::{self, moderation::add_channel_moderator};
let request = add_channel_moderator::AddChannelModeratorRequest::builder()
    .broadcaster_id("1234")
    .moderator_id("5678")
    .build();
let response: helix::moderation::AddChannelModeratorResponse = client.req_post(request, helix::EmptyBody, &token).await?.data;

You can also get the http::Request with request.create_request(&token, &client_id) and parse the http::Response with AddChannelModeratorRequest::parse_response(None, &request.get_uri(), response)
